# Парсер для получения структурированной информации с реестра (РКН) операторов, осуществляющих обработку персональных данных.

## Установка

`npm i`

## Запуск

`node index`

## Вход

Список ИНН для обработки списком в inn.txt

## Выход

Excel-таблица с полученной информацией от [https://pd.rkn.gov.ru/operators-registry/operators-list/](РКН).
